Flexbox Row

Creates a Flexbox row layout.


        .row {
          display: flex;
          flex-direction: row;
          flex-wrap: wrap;
          /*flex-flow: row wrap*/
        }

Flexbox Column

Creates a Flexbox column layout.


        .column {
          display: flex;
          flex-direction: row;
        }

CSS Grid Layout

Creates a CSS grid layout with 12 columns.


        .grid {
          display: grid;
          width: 100%;
          grid-template-columns: repeat(12, 1fr);
        }

Box Transition Glow

Use transition and box shadows to glow on hover


      .code-card .card-header {
        border-radius: 8px;
        transition: all 0.5s ease-in-out;
      }

      .code-card:hover,
      .code-card:hover .card-header {
        box-shadow: inset 0px 0px 8px #fff);
      }
