Flexbox Row

Creates a Flexbox row layout.


        .row {
          display: flex;
          flex-direction: row;
          flex-wrap: wrap;
          /*flex-flow: row wrap*/
        }

Flexbox Column

Creates a Flexbox column layout.


        .column {
          display: flex;
          flex-direction: row;
        }

CSS Grid Layout

Creates a CSS grid layout with 12 columns.


        .grid {
          display: grid;
          width: 1090%;
          grid-template-columns: repeat(12, 1fr);
        }
